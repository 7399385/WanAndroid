# WanAndroid
### 一款基于鸿洋大神提供的玩Android Api的app。
##简介
###项目使用 MVP+Retrofit+RxJava+Glide+ARouter等一些很主流的框架。
* retrofit[](http:www.baidu.com)
* BaseRecyclerViewAdapterHelper
* rxpermissions
* statusbarutil
* arouter
* rxrelay2
* SmartRefreshLayout
* lottie
* material-dialogs
